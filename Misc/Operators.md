## Operators

### Ternary Operator(statement ? obj : obj)
```javascript
let gender = user.male ? "male" : "female";
```

### Elvis Operator (?: )
```javascript
let displayName = user.name ?: "";
```

### Safe Navigation Operator (?.)
```javascript
let displayName = user.name ?. "";
```

### Logical or operators
```javascript
let displayName = user.name || "";
```
